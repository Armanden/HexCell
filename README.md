## HexCell

![HexCell)](screenshot000.png)

### Description

A game where a core cell collects other hexes/cells to add on its core and grow.
The core cell and hexes have atributes effecting their movements via wheit, health, 
and other effects that play a role in the merging mechenics as some hexes aren't able to merge with others.

### Build
To build the project the odin compiler and ofcourse raylib is needed, it compiles via the simple and humble: 
'''
odin build .
'''

### Features

 - Open World
 - Crystal like design

### Controls

Keyboard:
 - WASD to move
 - Enter to start and reset
